
# Cryptotick
Chrome extenstion for Cryptocurrency using coinmarketcap API build using yarn and react.

### Develop
#### Install packages
```sh
yarn install
```
#### One time build
```sh
yarn build
```
#### Watch files and rebuild
```sh
yarn watch
```
